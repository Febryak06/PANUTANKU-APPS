Unduh file mapping

Galeri State UI

Contoh loading, empty, success, dan error untuk memastikan feedback sistem selalu jelas.

Loading state

Sedang memuat ringkasan risiko... Mohon tunggu sebentar.

Empty state

Belum ada kasus aktif saat ini. Buat kasus dari hasil analisis untuk memulai tindak lanjut.
Empty state tetap mengarahkan ke aksi berikutnya.

Success state

Mapping komoditas berhasil disimpan. Data siap diteruskan ke modul analisis transaksi.
Gunakan state ini setelah pengguna menyelesaikan aksi penting.

Error state

File tidak dapat diproses. Periksa format dan isi file, lalu coba lagi.
Pesan error menjelaskan masalah sekaligus langkah lanjutannya.
